Download Manager Restrictions

The download manager enables you to manage your downloadable products. There are however, the following restrictions:

[1] - Only locally hosted files are supported. If you remotely host your downloads, use FTP to manage them.
[2] - PHP5 or higher with json_encode enabled is required.
[3] - As downloadable products should be stored outside of the web root, previews and downloads are NOT possible from within the download manager.
[4] - File uploads are restricted to the limit imposed on your server. For large files, FTP is always recommended.
[5] - Some command line functions may be required for some operations. If your server has them disabled the download manager may fail.

The following can be accomplished within the download manager:

[1] - New folders can be created. Existing folders can be moved to other locations via drag and drop.
[2] - You can upload multiple files to any folder.
[3] - You can rename files/folders and delete files/folders.

Other features are disabled as the download manager has been adapted for Maian Cart. If you find the download manager not suitable or you can`t get it to work, use FTP, such as FileZilla, which is free.
