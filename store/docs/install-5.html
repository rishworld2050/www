Character Set (Charset)

Charset when changing the language files is located in the following file: templates/language/english/global.php

Language

Apart from the e-mail templates, ALL language is found in the language files: templates/language/english/*.php
admin/templates/language/english/*.php Use a good text editor to edit. Notepad should be fine. BEFORE editing any of the language files read the notes in the following files: templates/language/README.txt
admin/templates/language/README.txt

E-Mail Template Language

All the language in the e-mails is in the e-mail template txt files: templates/language/english/email-templates/*.txt
admin/templates/language/english/email-templates*.txt Again, use a good text editor to edit.

Language Translations

If you don`t want to remove or edit the existing language files, but need to translate the files into another language, makes copies of the following folders and rename them with your new language. templates/language/english/
admin/templates/language/english Then in the settings, update the language to point to the new folder you have created.

Multi Language Cart

Its possible for you to have a multi language option in your cart. For this you need multiple language packs. Once this is in place, you can switch via the following: index.php?lang=english
index.php?lang=german
index.php?lang=finnish Where 'english', 'german' and 'finnish' are folders located in the 'templates/language/' folder. If this option is utilised, a cookie is set on the visitors computer so that their selection is remembered next time. Cookie auto expires after 30 days.

Product data is not multi lingual.

Language Packs

Available language packs can be found on the Maian Cart website. If you would like to send me a language pack, please use the contact option on the Maian Cart website.
