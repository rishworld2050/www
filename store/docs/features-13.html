MP3 Player

Maian Cart has an MP3 preview option for music items. The player used within Maian Cart is the brilliant open source MP3 Player from http://flash-mp3-player.net/ Check the website for further customisation help.

Player Customisation

The MP3 player is fully customisable and can adapted into your own site and colour scheme. See the website for further information. To edit the parameters view the following file:

templates/js/javascript.js

Look for the 'playMP3' function. This is global for all preview buttons.
