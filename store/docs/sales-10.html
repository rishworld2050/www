Invoice

Here you can view an invoice which can be printed and included with shipments.

Invoice Appearance

The invoice template files are not encoded, which means you can edit the layout if you want to and change colours. Applicable files to edit are:

admin/templates/windows/invoice.php
admin/invoice.css

Use a good text editor to edit, then re-upload. Make backup copies first.

Export to PDF

Its very easy to export invoices to PDF files if you want to keep copies. For this, a print server is recommended. When viewing an invoice in your browser, select the print option and the PDF print server. There are some excellent free ones around, including Bullzip, doPDF, CutePDF, PDF Creator & PDF24 Creator. (Note that I am in no way affiliated with these programs and they are used at your own risk)
