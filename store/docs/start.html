Overview

Sometimes with new software its hard to know where to get started. This is a list of the basic operations to get your store up and running. You should make sure you have installed the system first and configured your settings as per the installation instructions.

Step 1: Categories

Firstly, create some categories. Parent, children and infants are supported, so 3 levels.

Step 2: Add Products

Once at least one category has been added you can then add products to your store. It is recommended you familiarise yourself with the features on the add product page before attempting any batch import routines.

Step 3: Shipping Options

Shipping options should be set up next if required. Its not required that any shipping options be set. If no shipping zones are specified, checkout page is a more simpler option. If shipping does apply you should add countries and zones.

Note that various shipping overrides are in place throughout the system.

Step 4: Payment Methods

Payment options and methods should be set up next, depending on which ones you want to enable. Its a good idea to look at the Global Payment Settings first via 'General Settings > Global Payment Settings' in your administration area.

Payment methods should be set up next.

Step 5: Test Store

Once you have completed the above steps, your cart should be ready for a test. If things don`t work as expected, double check the main points above to make sure you followed the instructions correctly.
