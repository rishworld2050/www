Overview

From the 'Manage Products' screen you can esily manage products via the the options listed below. You can also filter products by category and search for products via keywords or product ID.

Enter product code, ID or keywords

To search your product database, enter the product code, id or a keyword from its title. The ID is found in the product url in the store. Examples:


Edit

Enables you to edit a product. Options here are shown in more detail on the Add Products page.

Copy

Pre-fills data and enables you to quickly copy an existing product. You can also copy attributess, pictures, mp3 previews, personalisation options & related products via the checkbox options.

Attributes & Attribute Groups

Product attributes are items related to the main product. Might be different sizes for clothes or different colour options. You can batch add attributes via the batch option or enter data in the boxes provided. You may also add multiple attributes at a time by using the + and - symbols to add / remove boxes.

More in depth information on the attributes feature here.

Pictures

Products can have unlimited pictures attached to them and be local or remote. Local images can be batch added and new folders can be created on the fly using the option provided. You can also select your main display image by using the radio buttons beneath the pictures. Your main display image is the one shown on category and product pages as the main image.

Supported local image formats: JPG  /  JPEG  /  PNG  /  GIF

Please resize your image to 800 x 640 @ 90 dpi before upload for best performance. Around 150KB max is recommended. A number of free image resizing utilities exist, so search on Google. Uploading large images will kill the the store load time for visitors and will impact on sales.

In the store front, the picture full size previews are controlled by colorBox. Use the option provided to create new picture folders if required.

NOTE: If you prefer NOT to have pictures auto renamed, see the F.A.Q page.

An alternative option is to have your images load from remote servers. If this is the case, you can upload a CSV list of image paths. CSV must NOT contain a header row and have 3 columns:

Image = Full http path to remote image.
Thumb = Full http path to remote thumb if applicable. Blank column if no thumb.
Display = Yes or no to determine main display image. If not specified first image will be main display image.

You can have as many image paths in the csv as you need. Deleting a remote image from the cart database won`t delete your remote image. Note that size orientation and dimensions won`t be shown for remote images. Also, copies of remote images are not saved locally.

MP3 Previews

Product mp3 previews. Unlimited mp3 previews are supported. To add MP3 files, upload .mp3 files into the 'templates/mp3/' folder. You can upload .mp3 files into this root folder or include them inside sub folders. Examples of structure:

templates/mp3/file1.mp3
templates/mp3/file2.mp3

Inside sub folders:

templates/mp3/my-music/file1.mp3
templates/mp3/ayumi hamasaki/file2.mp3

Once mp3 files are in place, refresh MP3 previews screen. Use the checkboxes to add mp3 files. You may assign mp3 files to a product from different mp3 folders. Enter a mp3 name/title if applicable. This is optional.

Related

Related products are exactly that, related. Related products appear on the product page of the product that it is related to. This is entirely optional.

If you mirror product relations, this would update across all selected products. For example if you mirror relations and product 4 related to products 1,2 & 3, products 1,2 & 3 would be updated to relate to 4.

Personalisation

Useful feature if you are selling gift items that can be personalised. You can specify a typed entry, or a list selection and you can have as many personalisation options per product as you need. You can restrict typed entries to a set amount of characters and also specify additional charges for each option.

More in depth information on the personalisation feature here.

View / Delete

The view option loads the store front display and the delete option clears the product and everything related to it from the system. This is NOT reversible.

One Click Enable/Disable of Product

To quickly enable or disable a product, click its product status. ie: ENABLED /DISABLED

Batch Edit Selected Products

Use this option to batch edit products. Check boxes first to determine which products to edit.

Warning!! Batch operations will apply to ALL selected products and is not reversible.

Backup your database first before any batch routines. Use the buttons to remove areas if you don`t want to update them. If none are excluded, all are assumed. If you refresh the page you may have to make your selections again.
