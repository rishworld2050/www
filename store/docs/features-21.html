elFinder File Manager

Maian Cart utilises the excellent elFinder File Manager, created by Dmitry (dio) Levashov. This enables you to manage downloadable products via a simple, yet effective file/download manager.

Restrictions in Maian Cart

Because product downloads are recommended to be outside of the web root, previews and downloads of files from the download manager interface are not available. It will also only work if your server has PHP 5 support with json_encode enabled. Errors will appear if you don`t.

Styling elFinder

All files that relate to the appearance/styling of elFinder are located in the following: admin/templates/css-elfinder/

elFinder Language Options

The language of the file manager is not controlled via the PHP language files. Instead its hard coded in the javascript files. English is default, but other available languages are located in: admin/templates/js/i18n/ To change to your own locale, first check which languages are available. You then need to udpdate the following in the 'admin/control/defined.inc.php' file. define('ELF_LOCALE', 'en'); Insert the 2 character language code ONLY. See the notes for an example.
