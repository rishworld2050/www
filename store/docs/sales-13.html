Overview

This feature lets you add a new sale to the system. This is useful if for example, you take an order by phone and want to add it to the system.

Add Products to Sale

Use this option to add products to a sale. Select a category to load products. Use the checkboxes to select the products you need to add. Once you`ve added products the screen will refresh. You can then adjust other areas for the product. Note that if the product is disabled it will not appear in the list.

Remove Products from Sale

To remove a product, simply specify 0 as the quantity. This will highlight the product. When you click 'Update Sale' it will be removed.

Toggle Attribute/Personalisation Options

If a product has attributes or personalisation options attached to it you will see toggle links relating to each or both:



Click the link to open/close the options for each. Once you have selected or updated which options to use, manually modify the totals on the right to match>



Once this is done you can click 'Re-Calculate Totals' and update sale.

Print Personalised Details

This option appears if at least one product has personalisation options. This provides a useful printer friendly version of the order personalisation.

Download Re-Activation History

For downloadable products only. Loads window where you can re-activate download links that have expired. Re-activation history is provided so you can see what was re-activated and when and which user actioned it.

You can also view the click history for a download by clicking 'View Click History'. This details download link clicks. This is useful if someone says they haven`t clicked a download link and had problems. Some people may try to get download links reset fraudulently, so double check.

The download window also enables you to lock access to the download page.

Re-Calculating Totals

If you have changed quantities or adjusted prices and or shipping options, clicking this link will re-calculate the totals. You can also adjust the totals manually if you prefer. When you click 'Update Sale' the totals will be saved. Ideally you need to refresh totals if you have changed quantities, specified to remove a product or adjusted personalisation options.

Note that the recalculation doesn`t re-calculate the correct shipping method. You should select the shipping method first before recalculation.

Edit Status & Notes

This lets you change the status after an edit and write an edit status. This is optional.

Help Tips

The same or additional information may be provided by hovering your cursor over the image next to each heading in the admin area if applicable.
