Overview

Product personalisation is a strong feature of Maian Cart. It enables you to sell personalised items, such as business pens & diaries or sell flowers where addresses are required. In the commercial version you can have an unlimited number of options per product. Note that personalised options apply to a product, not its attributes.

Personalisation Instructions

Instructions for visitor. For example, if you are selling products that can be personalised with someones name, the instructions might be:

Enter Your Name

You can also specify an alternative text to display in e-mails and on order pages, this is a second parameter separated with a pipe. Example:

Enter Your Name|Name

If the second parameter isn`t used, the main text is used. So, for example, lets say you are selling football shirts and people can pick the name they want on the reverse. You might specify:

Enter Preferred Name|Name

On the product pages, visitors would see "Enter Preferred Name", but in order e-mails and on the checkout page they would see the alternative of "Name". The second parameter MUST be separated with a pipe (|) and is optional. If the second parameter isn`t set, "Enter Preferred Name" would be used for all.

Max Characters

If a maximum character limit applies, visitors can only enter that amount of characters. For example, you might be selling business pens and diaries where the company name can be no more than 50 chars. Leave blank for no limit. If a limit is set, a counter appears so visitors can see how many characters they have entered.

Additional Cost

Is an additional cost incurred for this personalisation? If yes, enter value. No commas. This is added to the product total.

Leave blank, 0, or 0.00 for no costing.

Personalisation Options (Optional)

If the personalisation can only be one of a set of options, list them here. One per line. If this is set, max character limit is ignored. So, for example, you might be selling business pens that can be personalised with a certain colour, but only from select options. You might enter:

Blue
Green
Red
Yellow

Only these options would be presented to visitors.

Required Field

Is this a required field? If yes, the system will prompt a visitor if this personalisation option is not completed when adding product to cart.

Input Box Type

Which box type do you wish to display for this option? An input box is for single line entries, a textarea for multi line. If options are declared this is ignored.
