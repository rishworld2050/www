Overview

Enables you to add new products to the system. Batch importing is also supported, for further information see the 'Import Routines' page.

Product Name

This is the name of your product. Max 250 characters.

ISBN Lookup

Appears only if the ISBNDB API key is set in settings. More info on the ISBN lookup here.

To lookup number, enter book ISBN number in product name field.

Categories

Products can be in different categories if you wish. Use the checkboxes to specify which categories the product belongs to.

Product Brands

Product brands are an additional filtering option on the category pages. If a product belongs to a brand, specify the brand or brands. If you have the same brand created for parent and children categories you should specify the product to be attached to both brands.

Product Descriptions

SHORT Description - This is the product short description which appears on category pages. No formatting is allowed here and this can be alternative text to your full description or part of the same text. If some of your short text is long, you can restrict the view to keep things tidy via the 'Max Product Desc Characters on Category Pages' option in the settings.

FULL Description - This is your products full description which appears on the product page. Use BB Code for formatting, HTML is not allowed.

Use the copy options for quickness if you want to copy text between description fields.

Meta Keywords / Meta Description / Browser Title Text

Search engine meta data. Not as important as it probably once was, but its still seen as an important SEO technique by some. If these fields are left blank, the system will refer to the category meta descriptions instead. If category are blank, then the main site tags are used.

If you prefer different text to appear in the browser title bar than the product name, enter text in the title bar field. Use the copy option to copy product name if required. This is optional. If left blank, product name appears in browser title bar. Most modern browsers hide the title bar these days, but this may be useful for SEO techniques.

Product Tags

Specify product tags. These are keywords relating to the product. Can be good for search engines and also SEO techniques. Use the links provided to auto create tags from the description or meta fields.

Downloadable Product / Server Path to File / Download Limit

Specify if this product is a downloadable item. For example, a download link is offered after purchase for the buyer to download the file to his/her hard drive.

Local downloadable files should be placed OUTSIDE of the web root for safety. This will prevent them from being accessible to download managers.

[1] - Click the 'View Local Files' link to view your downloadable products and select the product applicable. Or enter file name manually. If the file resides in a folder inside your downloads base folder you need to include this too. Examples: file.zip
folder1/file.zip [2] - Alternatively, you can specify a remote url via http:// or ftp:// if your server supports the FTP functions. Useful if you have your downloads on remote servers.

HTTP links will be the full path to the remote file starting http://: http://www.myremotesite.com/downloads/file.zip To specify the FTP path, you must include your ftp username, password, host name and the file path. Lets say your remote file is located on a remote server at: public_html/downloads/file.zip The path you should specify to this file would be: ftp://USERNAME:PASSWORD@HOST/public_html/downloads/file.zip (Replace USERNAME,PASSWORD & HOST with your own details)

Another example might be: ftp://USERNAME:PASSWORD@HOST/yoursite.com/public_html/downloads/file.zip (Replace USERNAME,PASSWORD & HOST with your own details and replace yoursite.com)

It depends on your server structure and path. This may be trial and error. The path will be generally from the location of your ftp root details. So when you log in via FTP if you see a public_html folder the path should start with that. If you see a list of domains, it will start with the relevant domain. FTP files can be stored outside of the web root on the remote server as well.

The download limit is the amount of times each file can be downloaded.

Product Video

Product videos are to be uploaded into the 'templates/video/' folder. FLV is supported. Once uploaded, specify the applicable video from the drop down menu.

Enable Disqus Comments

Maian Cart uses the Disqus comments system. More info here.

Product Not Available In

Are there certain Countries that this product cannot be shipped to? If yes, check boxes against restricted Countries. Checkout won`t be available if a banned Country is selected.

Optional Text/Instruction on Checkout Page

Optional text/instruction which appears beneath the product title on the checkout page. Leave blank for no text..

Min Purchase Quantity

If minimum purchase quantity is set, buyer must purchase this quantity before they can checkout. Set to 0 to disable, which defaults to standard of min 1 item. Useful if you only sell volume amounts.

Stock / Low Stock Notify

Enter your stock level. For downloadable products you should specify a high amount, such as 999999. Or set the system to not reduce stock on downloadable items on the global payment settings page.

The low stock notification option enables you to receive e-mail notification if the product or any of its attributes reaches a certain stock limit. This can be a useful way of knowing when to order new stock levels.

Code/Model No

Product code number or identification code. A tick or a cross will appear when you start typing. A tick indicates the code is unique, a cross indicates a product is already using that code.

Weight / Price / Offer Price / Offer Expiry Date

Enter the product weight. This isn`t applicable if the product is downloadable, but is required for physical goods to determine shipping costs. This can be any kind of weight band, but must match your product weight. For example, if your products are in grammes, use a gramme range. ie: 0 to 1000.

Price and offer prices should be entered with no commas. ie: 9.99, 23.99, 1023.45 etc

If downloadable item it can be a free product by specifying 0.00

If you have set an offer price you can use the expiry option to have the offer revert back to full price on a certain date.

Enable Cart Purchase

Can this product be purchased in the store? If no, product still displays, but buy option is disabled. Useful if you want to advertise products available elsewhere and not sell them online.

Free Shipping

Do shipping rules apply to this product? If this is a downloadable product, shipping rules are not applied anyway.

Main Product Image

The main product image can be uploaded at the time of adding the product. This is optional as pictures can be added later. Recommended size is around 150KB. Images that are too big will take longer to upload and also take longer to load on your website.

Specify which folder to save main image to for this product. Create new folders inside the "templates/products" folder and give them write permissions.

Or click the button to add a new folder! Note that folders must be removed manually.

Additional Images

Up to 3 additional may also be added at the time of adding a product. This is optional. Pictures can be uploaded and managed later if you prefer.

Help Tips

The same or additional information may be provided by hovering your cursor over the image next to each heading in the admin area if applicable.
