Report a Bug

If you think you have found a bug in Maian Cart please let me know as soon as possible, so that it can be fixed. Before you report a bug, check to make sure your question hasn`t already been answered in the FAQ or on the support forum. What you *think* is a bug might not actually be one.

Report a Bug Procedure

If you wish to procede with a bug report, please send me the following information:

1. A description of what the bug is and how it can be replicated. If possible get some screen grabs if you see specific error messages.

2. If the bug is a database related issue, please let me know your PHP & MySQL server versions. Contact your host if you aren`t sure.

3. The script version number you are using. This is shown in your admin area and is also stored in the settings table in your database.

4. Please email the details to me here.
