Cloud Zoom

Cloud Zoom is a free and superb jQuery image zoom plugin. Besides the full size image preview provided by colorBox, Cloud Zoom provides a way for visitors to magnify and explore images, which can be useful as very large images are not easily seen in the browser viewing port.

Cloud Zoom Customisation

The Cloud zoom option is fully customisable. See the integration page for further information. To edit the parameters view the following file:

templates/js/cloud-zoom.js

Some colour options are set in the main stylesheet:

stylesheet.css

Note that this is best suited where you are adding good quality images which have produced a good quality thumbnail. Small thumbnails and images may product unexpected results. If you wish to disable the cloud zoom, open the 'control/defined.inc.php' file and set 'ENABLE_CLOUD_ZOOM' to 0.

Inner Zoom

If you prefer the zoom to load inside the image, instead of appearing to the right, open the 'templates/js/cloud-zoom.js' file in a text editor, and at the bottom of the file, change the position parameter to 'inside'.

position: 'inside',
