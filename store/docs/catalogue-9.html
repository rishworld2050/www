Overview

Enables you to add discount coupons to the system. This enables visitors to get discounts on products. Can be a useful marketing option.

Campaign Name

This is the name of your new campaign to group coupons into. Can be anything.

Usage Amount

Enter usage amount if coupon can only be used a set amount of times. 0 = unlimited.

Expiry Date

Enter expiry date if coupon is to expire on a certain date. Leave blank for no expiry. If usage amount and expiry is set, coupon expires when either reaches their limit first.

Min Spend Amount

This is the amount a customer must spend before they can apply a coupon code from this campaign.

NOTE: This does not apply if the Free Shipping or No Tax discounts are utilised.

Discount applies to total only, not shipping rate. Example:

9.99

Set to 0 for no minimum amount. It is recommended you set a minimum spend amount.

Discount Code

Code number for coupon. This is the code a visitor enters on the checkout page to get discount. Alphanumeric ONLY is recommended.

Discount

This can be any of the following:

2.99 - (Fixed amount discount off cost of goods)
5% - (Percentage discount off cost of goods)
freeshipping - (Offers Free Shipping ONLY on ALL orders - Min spend amount not applicable)
notax - (Offers No Tax Payment ONLY on ALL orders - Min spend amount not applicable)

Fixed & percentage discounts apply to sub total only, not shipping or tax. If free shipping is already in place, free shipping coupons are ignored. If no tax cost is already in place, free tax coupons are ignored.

A value must be entered in this field.

Free Shipping with Min Threshold

If you need an option to allow free shipping if a min spend amount is reached, look at 'General Settings' > 'Global Payment Settings' > 'Free Shipping Threshold'

Enable / Disable Coupons

Easily disables or enables a campaign without the need to delete it.

Coupon Report

Click the graph icon after adding coupons to see an overview of coupon usage for that campaign.

Help Tips
