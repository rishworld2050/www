What is MoneyBookers?

Moneybookers is an e-commerce business that allows payments and money transfers to be made through the Internet. It serves as an electronic alternative to traditional paper methods such as cheques and money orders. Moneybookers performs payment processing for websites, online auction sites, and other corporate users.

Moneybookers Payment Gateway

The Moneybookers payment gateway enables merchants to receive money from their customers via the Moneybookers website.

Enable Moneybookers Payment Gateway

1. Sign up for an account at Moneybookers if you don`t already have one.

2. Next, you must contact Moneybookers to enable their quick checkout option. To be approved for this and other extra services, please send an email to merchantservices@moneybookers.com with the following info:

- Your Website URL.
- The country in which you are located.
- Your Moneybookers account email.

Once approved, a new link will appear in your account area called 'Merchant Tools'.



3. Click 'Merchant Tools' and enter a secret word for your account:



4. On the same page, scroll down to 'Merchant Refunds'.



Check box to 'Enable Service' and from drop down select 'Website & API'. Maian Cart doesn`t currently support refunds from the API, but this may appear in a future version should the Moneybookers system be updated. The rest of the options on this page can be ignored.

Moneybookers is now ready to go.

Enable Moneybookers as Payment Option in Maian Cart

Select 'Catalogue > Payment Methods'. Click the icon and launch the Moneybookers configuration window. Complete the options shown to activate Moneybookers. More info as follows:

Merchant Account E-Mail

Your merchant account e-mail address for payments.

Secret Word

Secret word as set in your moneybookers account area on the merchant tools page as described above.

Information About this Payment Method

This is presented to visitors if they click for more info on the gateway on the checkout page. Optional. Use BB Code for formatting, HTML is not allowed.

Alternative Redirect Url (Optional)

Alternative redirect url is the page a visitor will be directed to after a successful transaction. You should NOT use this option if you sell downloadable items as visitors will not be able to download their products immediately after purchase.

Enter full url starting http://

Moneybookers Logo

Specify logo to load on payment page if required. Must be FULL url starting https://

If you do not have a SSL certificate installed, this MUST be left blank or else logo will NOT be shown.

Language

2-letter code of the language used for Moneybookers pages. Can be any of: EN, DE, ES, FR, IT, PL, GR, RO, RU, TR, CN, CZ, NL, DA, SV or FI

Check the Moneybookers site for further updates.

Live/Test Payment URL

Moneybookers processing url. At the time of these docs it is:

https://www.moneybookers.com/app/payment.pl

This should not be changed unless altered by Moneybookers.

Enable Moneybookers

Enable / disable Moneybookers as payment option.

Moneybookers Test Mode

To activate in Maian Cart, set the payment mode to 'Test' on the 'System > General Settings > Global Payment Settings' page. For testing you`ll need 2 Moneybookers accounts. A Consumer/Individual account and a Business/Merchant account. If you contact Moneybookers they will set up these accounts for you free of charge and give you more information.

Use the Individual account e-mail address when paying and the Business e-mail address as your account address in your settings. The Moneybookers system is very reliable and one of the easier gateways to implement.

Passing Additional Payment Parameters

Maian Cart allows you to specify additional payment parameters besides the required system ones. This maybe useful if you need to pass a certain parameter to any gateway during purchase. Some gateways have many parameters that enable/disable functions on the checkout page. To add addtional parameters for Moneybookers, enter array key => value pairs in the following file:

classes/gateways/inclusions/moneybookers.php

This is for advanced users.

Performing Additional Actions After Successful Payment

Maian Cart allows you to perform additional actions after a successful transaction. This is useful if you need to execute additional code after the payment has been accepted. Edit the following file:

classes/gateways/triggers/moneybookers.php

This is for advanced users.

Re-Transmitting API Messages

At the time this document was created, Moneybookers currently had no options to re-send post data back to the website.
