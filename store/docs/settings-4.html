Homepage Banners

A useful promotional feature is the homepage banner rotator. You can add as many banners as you like to the system and each banner can have an optional url to link to an internal or external site link. This can be useful if you are running different promotions during the year. Banners can also be easily enabled/disabled after being added.

Locate Image

Locate an image file on your hard drive. Images should be a valid image format like jpg,jpeg,png,gif.

Image sizes should ideally be 711px x 200px unless you have modified the main layout in any way and ALL images should be the same size.

Banner Description

Banner description is the text that appears in the images alt & title fields, so for cursor hover only.

Enter Url

Url is optional. If specified, banner is clickable. This can be an internal or external site url. Examples:

http://www.somesite.com (External to another site)

?pID=4 (Internal to store product page)

?pCat=9 (Internal to store category page)

http://www.yoursite.com/cart/?pID=4 (Also internal to store product page)

Homepage Banner Image Slider Options

The Maian Cart banner rotator is powered by a simple image slider written by Soh Tanaka. To change the timer duration, edit the following file:

templates/js/slider.js

If you add your images, be sure to adjust the div height/width in the main 'stylesheet.css' file. Look for the #slider directive.

Banner Ordering

To change the order the banners appear, hover your cursor over the banner rows until you see a crosshair. Click and drag to re-order.

Help Tips

The same or additional information may be provided by hovering your cursor over the image next to each heading in the admin area if applicable.
