Encoded Files

None of the store template files are encoded, neither are the invoice or packing slip templates in admin. The administration area is fully encoded, but colour changes can be made via the stylesheet. Maian Cart`s front end store can be adapted into any existing web layout. Read the instructions below carefully.

Before You Begin

Before you start to update the Maian Cart templates or colours, its a good idea to have the following:

[1] A good search program/utility. Sometimes you are looking for a specific div name and you can`t find it. A quick search on the 'templates' folder should find you things quicker. There are a few great free programs around, but I would recommend Super Finder XT or my own personal choice, File Locator Lite.

[2] Colours are the next problem. Knowing what the hex code is and where its located in any CSS or template file. ColorCop enables you to pinpoint a colour area pretty quickly, just drag the eyedropper over the colour you are looking for. Its also free.

[3] Patience. Any developer will tell you this is paramount. If you are getting frustrated, come back to it later.

[4] Backup the original installation first in case you need to revert back to it later.

Template Integration

Maian Cart utilises the Savant2/3 template engines. These make integrating the layout into your existing site relatively easy compared with none templated systems. The templates for the system are located in the templates folder. Other HTML has been separated from the PHP code and can be found in the templates/html/ folder.

Where you see PHP variables, these should NOT be removed. Example: <?php echo $this->CODE; ?>. See comments in templates for certain variables. The .htm template files have variable braces. ie: {brace}. These should NOT be removed either.

To integrate Maian Cart into your existing site design is going to require some basic knowledge of HTML. You`ll start by looking at your current HTML structure and comparing it to that of Maian Cart. Generally work with the header and footer first. If you don`t understand HTML, integrating into another site layout is NOT recommended. It is also NOT recommended you integrate the layout using PHP includes.

Note that the admin templates are encoded and cannot be changed. Admin stylesheet and images may be changed/updated.

Header & Footer Files - Free v Commercial

The main difference between the commercial and free version footer files is that the closing area of the commercial version footer is encoded in the script. You should NOT attempt to bypass the loading of the footer file by using PHP code. Doing this may return unexpected results and cause the system to fail.

If you have been kind enough to pay for the commercial licence, you can edit the footer html via the following file:

templates/html/footer-commercial.htm

Or you can simple specify your copyright code in the settings instead of editing the html.

The header template file (header.tpl.php) is the same for both versions.

E-Mail Templates

Besides HTML templates, Maian Cart also makes it easy to modify e-mail data. The e-mail templates are located at:

templates/language/english/email-templates/*
admin/templates/language/english/email-templates/* (admin e-mail templates can be edited)

You can open these templates and add whatever text you want. You should be very careful NOT to remove any code between braces. ie: {name} Removing these variables will cause some data not to be included in e-mails. If you want to use HTML in emails, enable the option in the "control/defined.inc.php" file and use HTML code in the .txt files.

define('ENABLE_HTML_IN_EMAILS', 1);

Relevant gateway template files are separated into individual folders. More info on the email templates here.

Colours, Appearance & Images

For many people, changing the colours and a few default images is good enough. Maian Cart utilises Cascading Stylesheets for colour display. ALL of the colours you see for the default design are in the stylesheets. Maian Cart has the following stylesheets, all of which can be edited in a good text editor.

stylesheet.css (Frontend)
ie6.css (Adjustments for Internet Explorer 6 ONLY)
admin/ie6.css (Adjustments for Internet Explorer 6 ONLY)
admin/ie7.css (Adjustments for Internet Explorer 7 ONLY)
admin/menu.css (Admin Menu)
admin/stylesheet.css (Admin)
admin/greybox.css (Popup Window Styling Only)

Edit these and choose whichever colours you want. Images are located in the following folders:

templates/images/ (Frontend)
admin/templates/images/ (Admin)

LOGO: If you want to update the main Maian Cart logo update the 'templates/images/logo.gif' file. OR use the file upload option in the admin settings.

CSS3: The new CSS3 spec allows elements to have rounded corners. You may find that a few elements appear square in older browsers. Older browsers do not support CSS3. In most cases rounded images are used for compatibility.

Admin Invoices & Packing Slip Layouts

If you wish you can change the look and layout of the invoice and packing slip layouts in the admin area, these files are NOT encoded. Template files are located at:

admin/templates/windows/invoice.php
admin/templates/windows/packing-slip.php

Stylesheets are located at:

admin/invoice.css
admin/packing-slip.css

ColorBox

Maian Cart utilises the excellent jQuery plugin colorBox for certain display options. For example, when viewing a fullsize product image or description.

The colorBox has a fixed size (700 x 500) in Maian Cart to prevent the display from disappearing off the page. If you wish to change this, edit the following files:

templates/js/colorbox.js
admin/templates/js/colorbox.js

Set 'innerWidth' & 'innerHeight' to false to remove or enter alternative values.

For the payment gateway help windows, an override of 700 x 450 is in place. This can be changed in the 'templates/language/english/checkout.php' language file. Within this file, search for 'colorbox'.

Stylesheets for the colorBox divs/windows are located at:

colorbox.css
colorbox-ie.css
admin/colorbox.css
admin/colorbox-ie.css

Further options are available, including theme changes and styling. For more information on colorBox, click here.

MP3 Preview Player

Maian Cart utilises the Open Source Music Flash Player software for mp3 previews. This is a free player and I`m truly grateful for the authors hard work in making this great player available.

The file that loads the flash player track previews is:

templates/html/products/product-mp3-preview-track.htm

Parameters are loaded and can be customised in:

templates/js/javascript.js

The player is fully customisable via flash parameters, so just edit the above file are you are good to go. For more information click here.
