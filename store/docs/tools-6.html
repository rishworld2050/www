Overview

Displays log entries. Useful for security. Filter by users if required. To clear/export entries by specific user only, select the user first before clicking the clear and export links. Entry log is enabled/disabled in the settings.

Disable Logging of Users

Its possible to enable the entry log, but not log entries by certain users. For example, a site developer or owner might not want his/her log entries to appear.

To disable users, open the 'admin/control/defined.inc.php' file in a text editor and look for the following:

define('ENTRY_LOG_SKIP_USERS', '');

Enter usernames separated with a comma. This is not case sensitive. So, for example, to disable logging of 'johndoe' and 'ayumihamasaki', do:

define('ENTRY_LOG_SKIP_USERS', 'johndoe,ayumihamasaki');
