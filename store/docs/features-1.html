What is Ajax?

Ajax, shorthand for Asynchronous JavaScript and XML, is a group of interrelated web development methods used on the client-side to create interactive web applications. With Ajax, web applications can retrieve data from the server asynchronously in the background without interfering with the display and behavior of the existing page. In short, it enables data to be sent to the browser and dynamically altered without page refreshes. Text © Wikipedia

Ajax Checkout

Maian Cart uses Ajax directly with XML to send/receive data on the server. This is utilised to maximum effect in the system on the checkout page. Instead of loading multiple pages, all the checkout is loaded on a single page. This is with the exception of certain payment methods that require additional information. The javascript/ajax routines are not encoded in the system and can be viewed in any text editor. They are located in 'templates/js/' folders and should not be altered in any way.
