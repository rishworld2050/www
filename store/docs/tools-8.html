Overview

The front end of your store contains a newsletter signup box for people to subscribe to your newsletter and also an optional opt-in feature on checkout. Maian Cart does have a built in mailer, but for large databases it is recommended you use a proper batch mailing software as detailed below. Note that some hosts may disable your account if you are on a shared server sending out lots of emails.

SMTP MUST be enabled to use the built in mailer.

Managing Addresses

Click on an e-mail address to load an edit box, then click 'Update' to update the record. Use the delete options provided to delete e-mail addresses. Export and clear functions should be self explanatory.

Import/Add E-Mail Addresses

You can add single email addresses or batch import addresses if you wish. If batch importing, file must contain one email address per line. Either batch file or single email address must be specified or a combination of both.

Duplicates, if detected, will be ignored as will anything the system thinks is not a valid email address.

Prevent Spam Bots

If you are not using the newsletter option and have removed the code, you should also disable the possibility of bots auto sending post data. To do this open the 'control/defined.inc.php' file and set the following constant to 0:

define('NEWSLETTER_SIGNUP', 0);

Unsubscribe/Opt Out Link

To provide an unsubscribe link for buyers, use the url to your cart index.php file with the 'optOut' var appended. The value of this would be the visitors email address:

http://www.yoursite.com/cart/index.php?optOut=joe@testdomain.com
http://www.yoursite.com/cart/index.php?optOut=faye@testdomain.com

Most good mailing software lets you create mail merge variables, so you can auto create the links. Visitors can also of course use the box on the website.

Built in Batch Mailing System

Maian Cart has a basic batch mailer. On the 'Newsletter' page, click 'Send Batch E-Mail'.

It enables you to send batch emails with attachments. Attachments should NOT be any attachment that would be rejected by the incoming mailbox or anti virus software. ZIP and/or PDF are recommended. 3 Max.

HTML Message - This is your email in html format. The default template loads from the 'admin/templates/language/*/default-newsletter/' folder. Edit if required. Paths to images in the message should be FULL urls starting http:// or else the images will NOT load in the incoming email program.

Plain Text Message - Many people disable html emails for security. Because of this you should also provide a plain text version of your messsage. The default template loads from the 'admin/templates/language/*/default-newsletter/' folder. Plain text messages should contain no formatting. Hyperlinks will be rendered properly by the email program.

{unsubscribe} = Use this to auto create an unsubscribe option in your message.

SMTP MUST be enabled to use the built in mailer.

Finally, this operation may be slow if you have thousands of addresses. For large databases, use custom software properly designed for batch mail.
