What is a zip file?

The ZIP file format is a data compression and archive format. A ZIP file contains one or more files that have been compressed to reduce file size, or stored as-is. The ZIP file format permits a number of compression algorithms. Text © Wikipedia

Zip Functions in Maian Cart

This is implemented on downloadable items and enables all downloads from an order to be consolidated into a single zip file via PHP`s built in zip functions. Zip files are created on the fly and use no server space.

For this to work your server is required to have the PHP Zip Functions enabled and have support for the ZipArchive class. PHP5+ or higher is required.

Options for the zip functions are found in the administration area on the 'General Settings' page.

IMPORTANT! This feature will NOT work for remote files. If any downloads in an order are remotely located, this is disabled.
