What is AlertPay?

AlertPay is a free to join internet payment service allowing you to send and receive money online. It is a Canadian based company that was established June 19, 2005. Its a comprehensive all-in-one solution for your payment needs. AlertPay is an account-based payment processor allowing just about anyone with an email address to securely send and receive money with their credit card or bank account.

IPN: Instant Payment Notification

IPN stands for Instant Payment Notification. It is a notification message that contains all the payment details data as well as your customer's information. IPN messages are sent when payments are made on your website through our checkout page or when you have successfully paid someone through AlertPay's MassPay API.

Enable AlertPay IPN

1. Sign up for an account at AlertPay if you don`t already have one.

2. Log into your AlertPay account area and click 'Business Tools' from the menu:



3. Click on 'IPN Setup'. You`ll then be asked to enter your transaction PIN number:



4. When the IPN options load, you`ll see a screen like the following:



5. Into the 'Alert Url', enter the full url to the 'checkout-alertpay.php' file in your cart root. Example:

http://www.yoursite.com/cart/checkout-alertpay.php (Replacing the domain/path with your own correct path)

6. IPN Status should be 'Enabled'.

7. Enable 'Test' mode only if testing system.

8. Set a IPN Security Code.

9. DO NOT enable v2 IPN. Other options should also be ignored.

10. Click 'Update' to finish.

Alert Pay is now ready to go.

Enable AlertPay as Payment Option in Maian Cart

Select 'Catalogue > Payment Methods'. Click the icon and launch the AlertPay configuration window. Complete the options shown to activate AlertPay. More info as follows:

Merchant E-Mail Address

Your AlertPay merchant e-mail address for payments.

IPN Security Code

Enter security code as seen on IPN setup page in your AertPay account area.

Information About this Payment Method

This is presented to visitors if they click for more info on the gateway on the checkout page. Optional. Use BB Code for formatting, HTML is not allowed.

Alternative Redirect Url (Optional)

Alternative redirect url is the page a visitor will be directed to after a successful transaction. You should NOT use this option if you sell downloadable items as visitors will not be able to download their products immediately after purchase.

Enter full url starting http://

Live/Test Payment URL

AlertPay`s processing url. At the time of these docs it is:

https://www.alertpay.com/PayProcess.aspx

This should not be changed unless altered by AlertPay.

Enable AlertPay

Enable / disable AlertPay as payment option.

AlertPay Test Mode

To activate in Maian Cart, set your account IPN mode to test in your AlertPay account area. AlertPay no longer supports activating this from third party software, it MUST be done via your account area.

Passing Additional Payment Parameters

Maian Cart allows you to specify additional payment parameters besides the required system ones. This maybe useful if you need to pass a certain parameter to any gateway during purchase. Some gateways have many parameters that enable/disable functions on the checkout page. To add addtional parameters for AlertPay, enter array key => value pairs in the following file:

classes/gateways/inclusions/alertpay.php

Performing Additional Actions After Successful Payment

Maian Cart allows you to perform additional actions after a successful transaction. This is useful if you need to execute additional code after the payment has been accepted. Edit the following file:

classes/gateways/triggers/alertpay.php

This is for advanced users.
