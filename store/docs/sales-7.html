Order Update

Here you can update the sale with a new status and e-mail the buyer to let them know the status has changed. You can include additional e-mail attachments if you want to. The statuses help keep a running inventory of the progress of orders and when certain admin actions are performed.

Automatic Updates

Automatic updates are written when any of the following actions occur:

- New order received.
- Order is refunded (Paypal, Google Checkout & 2Checkout only).
- Download page is locked / unlocked (Can be turned off in 'admin/control/defined.inc.php').
- Downloads are re-activated (Can be turned off in 'admin/control/defined.inc.php').
- Products are added to sale (Can be turned off in 'admin/control/defined.inc.php').

E-Mails

Send E-Mail to Buyer - Check box to send e-mail to buyer.
Copy E-Mails - If you want to copy the e-mails to other addresses, enter them in the box provided separated with a comma.

Attachments

Up to 3 attachments can be sent with each e-mail. This is useful if you want to e-mail PDF invoices, tracking info or other documents.

If you want to keep a record of the attachments you can elect to save them to your server. For this the 'admin/attachments/' folder must be writeable. Use the option provided to create new folders. If folders can`t be created by the system they can be created manually.

Printer Friendly Version

Click the link to view printer friendly version of statuses if you need to print them out.

Saving Status Text Input

A time saving mechanism is to save status texts. This is useful if you find yourself typing the same messages for multiple orders. After you have typed the message in the box you can save that text for recall on other orders. Look for the following icons above the text area on a sale update page:

- Click to save textarea status text. Enter reference in pop up message and save.
- Click to load status text options. From drop down menu, select status to populate textarea.
- Click to launch status management screen to edit/delete statuses.

Edit Existing Status Text

If you need to edit any existing status, click anywhere on the status to open an edit box. Click outside of the box to update.
