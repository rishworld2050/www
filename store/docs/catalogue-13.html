Attributes & Attribute Groups

Each product can have associated attributes. Attributes are listed together in groups. For example, if you are selling laptop computers, you might have the following attribute structure: Screen Size
12"
15"
17.3" Operating System
Vista
Win7
WinXP Extras
Carry Case
Logitech Mouse
Transcend 8GB Flash Drive

Visitors can select multiple attributes when adding item to basket. One from each group if multiple option is specified or else its one per group.

Adding Attribute Group



Enter new group name, select multiple option and then click the + button to add group.

Allow Multiple Selections: Can visitor select multiple options from this group? If yes, more than one option can be selected on product page when adding item to cart.

Required Selection: If required, visitors must choose at least 1 option from the selected group.

Adding Attributes to Group



Click +/- to toggle boxes and select a group name to add attributes to.

Re-Ordering Groups



Use drop down menus to re-order groups.

Editing & Deleting



Use icons to manage attributes.

Copy Attributes

The copy option is useful if you have other products that have the same attributes. Click the link to batch add attributes via copying.

Batch Import Product Attributes

The import option enables you to batch add attributes. This can be a huge timesaver. Click the link to batch add attributes via importing.
