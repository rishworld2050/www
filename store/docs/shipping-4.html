Overview

Weight based hipping rates are the rates you offer for products in a certain weight band for a certain service. This can be any kind of weight band, but must match your product weight. For example, if your products are in grammes, use a gramme range. ie: 0 to 1000. Weight isn`t shown in the cart or on the checkout page.

Weight based shipping rates are not required.

Weight From / Weight To

Weight range. First rate should start 0. To specify only a single rate, enter a high amount for the 'from' field. This can be any kind of weight band, but must match your product weight. For example, if your products are in grammes, use a gramme range. ie: 0 to 1000. If pounds and ounces you might start 0 to 5.

Shipping Cost

Shipping cost for this rate.

Apply to Service

Which service(s) does this rate apply to? Use the checkboxes to batch add rates to multiple services if in add mode. If in edit mode, use select option.

Batch Adding Rates

Use the +/- buttons to add rows to batch add more than one rate at a time.
