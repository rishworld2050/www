1. This directory is for the 'System > Manage New Pages' page in admin.

2. Create files here with a .tpl.php extension.

3. Specify file as the load file when adding/editing a new page.

4. This will override the system default file and enable you to add in your own PHP for different pages.

5. If you want to parse BBCode on a custom template page you can use the following function:

   mc_txtParsingEngine(var);

6. For loading of homepage banners or category, see notes in 'example.tpl.php' file
